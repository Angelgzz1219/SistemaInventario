# Sistema de Gestión de Inventarios

Este proyecto es una aplicación desarrollada en Java que implementa los conceptos de programación orientada a objetos, incluyendo herencia, polimorfismo, clases abstractas e interfaces.

## Descripción

El sistema permite gestionar diferentes tipos de productos como electrónicos, alimentos y ropa. Cada tipo de producto tiene características específicas, pero todos comparten atributos comunes gracias al uso de una clase abstracta.

Además, se implementa una interfaz para calcular el precio final de los productos, demostrando el uso del polimorfismo.

## Clases del proyecto

- Producto (clase abstracta)
- IVendible (interfaz)
- Electronico
- Alimento
- Ropa
- VentanaInventario (interfaz gráfica)
- Main (clase principal)

## Funcionalidad

El usuario puede:

- Ingresar nombre del producto
- Ingresar precio
- Seleccionar tipo de producto
- Ingresar datos adicionales
- Visualizar el precio final

## Tecnologías utilizadas

- Java
- Java Swing
- NetBeans
